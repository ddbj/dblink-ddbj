# dblink-ddbj
This scripts creates the tables related to dblink such as BioProject, BioSample, DRR, and Accession number of DDBJ records. Regarding each table file, refer to [https://raw.githubusercontent.com/ddbj/dblink-ddbj/refs/heads/main/README_dblinkddbj.txt] file

See https://ddbj-dev.atlassian.net/wiki/x/KADVkg for installation.
